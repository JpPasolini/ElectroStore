# ⚡ ElectroStore

Aplicativo mobile desenvolvido em React Native com Expo, consumindo a [Fake Store API](https://fakestoreapi.com).

## 🚀 Como rodar o projeto

### Pré-requisitos
- Node.js instalado ([nodejs.org](https://nodejs.org))
- Expo Go no celular ([Android](https://play.google.com/store/apps/details?id=host.exp.exponent) / [iOS](https://apps.apple.com/app/expo-go/id982107779))

### Passos

1. Clone o repositório:
```bash
git clone https://github.com/SEU_USUARIO/electrostore.git
cd electrostore
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o projeto:
```bash
npx expo start
```

4. Escaneie o QR Code com o **Expo Go** no celular.

---

## 👤 Usuários disponíveis para login

Acesse no navegador:
```
https://fakestoreapi.com/users
```
Use o `username` e `password` de qualquer usuário listado para fazer login.
